Features

  • Keyboard-controlled piano interface – Physical keyboard keys are mapped to musical notes.
  • Polyphonic audio playback – Multiple notes can be played simultaneously.
  • Sample-based sound engine – Each note triggers a pre-recorded audio sample.
  • Synchronized video feedback – The most recently pressed note controls the video.
  • Interactive visual UI – On-screen piano keys respond visually.
  • Browser-based execution – Runs entirely in the browser.
