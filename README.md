##Interactive Piano Video Instrument

This project is a web-based interactive musical instrument that combines keyboard input, polyphonic audio playback, and synchronized video feedback.
Users can play a piano-like interface using their physical keyboard, triggering corresponding audio samples and visual responses in real time.

The instrument is designed as a sample-based polyphonic player, allowing multiple notes (chords) to be played simultaneously while displaying a single representative video for visual clarity.

Sample-based sound engine

Synchronized video feedback
A video corresponding to the most recently pressed note is displayed while notes are active.

Interactive visual UI

Browser-based execution
Runs entirely in the web browser without any additional software installation.
